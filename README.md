# Bank Marketing Campaign Analysis

## Overview
Customer segmentation and predictive response modeling on 4,521 bank customers 
using machine learning techniques.

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Power BI

## Key Results
- Segmented customers using K-Means and Hierarchical Clustering
- Built 3 classification models: Logistic Regression, Decision Trees, Random Forest
- Random Forest achieved highest accuracy (evaluated via ROC-AUC, precision, recall)
- Identified account balance, loan status, and campaign history as top predictors
- Handled 88% class imbalance in target variable

## Dataset
UCI Machine Learning Repository — Bank Marketing Dataset  
4,521 records | 17 attributes
